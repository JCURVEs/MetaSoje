# 메타버스를 활용한 소제동 보존 프로젝트
:calendar: 2022.9.-2022.12.
## Description
대전광역시의 새로운 핫플레이스로 등극하는 소제동에는 역사적 가치가 있는 도시이다.  

철거되는 관사촌을 기록하며 시대에 따른 여러 모습의 소제동을 제공할 필요가 있다.  

유니티를 이용해 소제동을 보존하고자 건축공학도가 모여 만든 프로젝트이다.
